でも、その中、コロナという病気が全世界に広がって私がしようと思ってた事が出来なくなりました。私は何をすればいいんだろうと沢山の考えの上、 勉強して来た言語とパソコンの言語という接点を通じてウェブ開発者として勉強をし始めました。

HOW TO STUDY

色んなパソコンの言語の中でJAVAが一番使われてると思い、カンナムのある塾に行って一度学んでみようとして略2ヶ月くらい学びました。そこでJAVAを利用し、色んな機能を作ってみました。 でも、目に見えない機能に段々これで何を作れるかと疑問が出来たと思います。

それでまず目にみえるFrontの方を勉強しようと思い、生活コーディングという無料の講義サイトを発見してそこでHTMLとCSSとJavaScriptの基礎を学べました。
その後Frontの方をもっと勉強したいと思い有料講義サイトで少し深化なHTML, CSS, JavaScriptを勉強し、またReactというライブラリーを勉強しました。

最近はFront と Back 全ても利用してウェブを作りたいと思ってJSP, JAVA, MySQLを勉強してログイン機能や掲示板を作ってみました。

Daily Japanese

  • 言語 / 技術
    HTML / CSS / JavaScript / jQuery / Java / JSP / MySQL
  • 参加度
    一人作り
  • 重点
    反応型ウェブ、アニメーション、ログイン、会員加入(Email認証)、掲示板(いいね機能、整列機能)
  • サイトの目的
    友達に日本語を教えた事がございます。この時の資料を記録と日本語を勉強する方達に少しでも力になれるサイトを作りたかったです。
    そしてその人達と日本について疎通したい思いで作ったサイトでございます。
1. 反応型ウェブ / アニメーション
  • BootStrapとMediaQuaryを利用して反応型ウェブで作りました。
  • Keyframeを利用してボタンのアニメーションを作り、CSSの3D(Flip)を利用して裏面が見えるように作りました。
  • Navを下にスクロールしても見えるようにFixedし、隠される内容もよく見えるようにスクロールされる際Navのサイズを小さくして暗くて透明になるように作りました。
  • Caroselをライブラリーを使用せず直接JavaScriptを利用して作ってみました。
2. ログイン / 会員加入(Email認証)
  • MySQLに繋げて会員加入とログイン機能を作りました。
  • ログインの状態にはログアウトだけ見えるように作りました。
  • 会員加入の際ID,Password,EmailにはRegular Expressionを使い、Regular Expressionに合わないとお知らせが出るようにしました。
  • Email認証の機能はSHA(Secure Hash Algorithm)を利用して保安を考えました。
  • 文書きの際ログインとEmail認証のした上、できるように作りました。
  • ログインはしましたがEmail認証が出来てない際にはEmail再発送ページに移動されるように作りました。
3. 芳名録 / 申告の機能
  • MySQLに繋げて芳名録を管理できるように作りました。
  • いいね機能を追加してMySQLを利用して推薦順整列でも並べるように作りました。
  • いいねは一回だけ押せるようにuserIDをMySQLに一緒に登録するようにしました。
  • 芳名録は５個ずつ見れるようにMySQLを利用して分けてみました。
  • MySQLを利用してTitleとContentを選んで検索出来る機能を作りました。
  • 申告機能を作って利用者達にメールをもらい、不適切な内容を早く管理できるように作りました。
  • 文を書く際JavaScript文法を書けないようにreplaceAll() を使用して保安を考えてみました。
