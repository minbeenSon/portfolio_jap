MOTIVATION

以前、私は空港の近くにあるホテルで働きました。その中で自然と私と住んでいた環境が全く違う人達と触れ合う機会に恵まれ、外国人に対して興味と関心が生まれ、愛情を抱くようになりました。 そして、私は様々な人と疎通する仕事をしたいと思い、言語の学習を始めました。
しかし、そんな中コロナウィルスが全世界に広がり、私がしようと思っていた仕事が出来なくなりました。
私は何をするべきか沢山考えたのち、
今まで勉強して来た言語とパソコンの言語という接点を通じてウェブ開発者として勉強をし始めました。

HOW TO STUDY

様々なパソコンの言語の中でJAVAが一番使われてると思い、カンナムにある塾に通い初めて約2ヶ月程学びました。
そこでJAVAの文法を学び、JAVAを利用してあらゆる機能を作っていくうちに、段々と目に見えない機能に対して、これで何を作れるだろうか、と疑問が出来ました。

それで、まずは目に見えるFront-endを勉強してみようと思い、生活コーディングという無料の講義サイトを発見してそこでHTMLとCSS、JavaScriptの基礎を学びました。
その後Front-endの知識をより深めたいと思い、有料講義サイトで更に深いHTMLとCSS, JavaScriptそしてjQueryを学習しました。
また、Reactというライブラリーを勉強しました。

最近はFront-endとBack-end全てを利用してウェブを作りたいと思い、JSP, JAVA, MySQLを勉強してログイン機能や掲示板を作ってみました。

Daily Japanese

  • 言語 / 技術
    HTML / CSS / JavaScript / jQuery / Java / JSP / MySQL
  • 参加度
    一人作り
  • 重点
    反応型ウェブ、アニメーション、ログイン、会員加入(Email認証)、掲示板(いいね機能、整列機能、検索機能)
  • サイトの目的
    友達に日本語を教えた事がございます。この時の資料を記録したいし、日本語を勉強する方達に少しでも力になれるサイトを作りたかったです。
    そしてその人達と日本について疎通したい思いで作ったサイトでございます。
1. 反応型ウェブ / アニメーション
  • BootStrapとMediaQuaryを利用して反応型ウェブで作りました。
  • Keyframeを利用してボタンのアニメーションを作りましたし、
    CSSの3D(Flip)を利用して裏面が見えるように作りました。
  • Navをどんな位置でも見えるようにFixedしました。
    隠される内容が少しでもよく見えるようにスクロールされる際、Navのサイズを小さくして暗くて透明になるように作りました。
  • Carouselをライブラリーを使用せず直接JavaScriptを利用して作ってみました。
2. ログイン / 会員加入(Email認証)
  • MySQLに繋げ、会員加入とログイン機能を作りました。
  • ログインの状態にはログアウトのボタンだけ見えるように作りました。
  • 会員加入の際、IDとPassword、EmailにはRegular Expressionを使い、Regular Expressionに合わないとお知らせが出るようにしました。
  • Email認証の機能はSHA(Secure Hash Algorithm)を利用して保安を考えました。
  • 文書きの際、ログインとEmail認証をした後できるように作りました。
  • ログインはしましたがEmail認証が出来てない際にはEmail再発送ページに移動されるように作りました。
3. 掲示板 / 申告の機能
  • MySQLに繋げ、掲示板を管理できるように作りました。
  • いいね機能を追加して推薦順の整列で並べるように作りました。
  • いいねは一つのIDに一回のみ押せるようにuserIDをMySQLに一緒に登録するようにしました。
  • 掲示板は５個ずつ見れるようにMySQLを利用して分けてみました。
  • MySQLを利用してTitleとContentを選んで検索出来る機能を作りました。
  • 申告機能を作り、利用者にメールをもらって不適切な内容を早く管理できるように作りました。
  • 文を書く際、JavaScript文法を書けないようにreplaceAll()を使用して保安を考えてみました。
